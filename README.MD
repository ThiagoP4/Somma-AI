# Somma AI - Personal Finance Dashboard

Projeto de gestão financeira pessoal focado em automação e análise de dados.

## 🚀 Tecnologias

- **Frontend:** Vue 3, TypeScript, Vite, ApexCharts, Phosphor Icons.
- **Backend:** Node.js, Express, Prisma ORM.
- **Database:** PostgreSQL (Supabase).
- **Automation:** n8n (AI Agent).

## 📦 Estrutura do Projeto

Este é um monorepo contendo:
- `/client`: Aplicação Frontend (Vue.js)
- `/server`: API Backend (Node.js)

## 🔧 Como rodar

1. Instale as dependências: `npm install` (na raiz, client e server)
2. Inicie o backend: `cd server && npm run dev`
3. Inicie o frontend: `cd client && npm run dev`
